# Phishing Email Analysis – Greenholt Phish (TryHackMe)

## 🛡️ Overview
This project documents my hands-on analysis of a phishing email using a virtual lab environment.

## 🔍 Key Tasks Performed
- Analyzed email headers to identify originating IP address
- Investigated SPF, DKIM, and DMARC records
- Identified suspicious Return-Path domain
- Extracted malicious attachment from email
- Generated SHA256 hash of attachment for malware identification

## 🧪 Tools Used
- Linux Terminal
- sha256sum
- TryHackMe VM
- Email header analysis techniques
